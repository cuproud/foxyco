Branch: `m6-showroom`. Milestone: full dark-UI redesign + garage system +

## What shipped

M6 turned FoxyCo from the cream "receipt" look into a dark green-black
"showroom" and added a multi-vehicle garage. Delivered across 11 tasks:

- **T1 — Dark tokens + theme flip.** `tokens.dart` reworked to a deep
  green-black surface stack (`bgBase/bgSurface/bgSurface2`), cream-on-dark text
  tiers, orange glow shadows, motion durations. `app_theme.dart` flipped to
  `ThemeData.dark`. Verdict colors kept, bright tier is now primary.
- **T2/T3 — Garage domain + controllers.** `lib/domain/garage.dart` (Vehicle +
  Garage model, v1 migration from the legacy single-profile shape).
  `garage_controller.dart` (garage + driver-name controllers). Old
  `profile_controller.dart` deleted, folded into the garage controller.
- **T4 — VehicleArt.** `lib/ui/theme/vehicle_art.dart` — layered CustomPainter
  art for 6 body types + fuel badge, driven by `bodyType/color/fuelType/width`.
- **T5 — Hero profile card rebuild.** `profile_card.dart` — time-of-day greeting
  bands (incl. "Late shift"), active-vehicle line, VehicleArt silhouette,
  sheen sweep.
- **T6/T7 — Slide-to-live + status polish.** `slide_to_live.dart` (drag-to-commit
  control replacing the Start/Stop button; medium haptic on commit, spring-back
  on cancel, reduced-motion aware). Home status card: monogram badges, count-up
  numbers, animated segment bar, glow.
- **T8 — Garage UI + editor + settings restyle.** `settings_screen.dart`
  restyled dark; `vehicle_editor_screen.dart` new full-screen route (via
  go_router `/vehicle-editor`); garage list with add/set-active/edit/delete.
- **T9 — History count fix + restyle.** `history_screen.dart` — header now counts
  the *filtered* range, smart empty state ("N offers outside these filters" +
  Show-all reset), dark restyle.
- **T10 — Splash.** `splash_screen.dart` — cold-start wordmark fade + car
  drive-in (~1.8 s), reduced-motion static path, crossfade to Home.
- **T11 — This sweep.** Validation, inline-color audit, overlay-untouched audit,
  manual-test rows, this doc.

## Behavior changed

- App is dark everywhere; cream is now text, not background.
- Start/Stop button replaced by a slide-to-go-live control (watch semantics in
  `dashboard_controller.dart` unchanged — only the trigger UI changed).
- Single profile → garage of vehicles; existing users migrate automatically
  (their one vehicle becomes the active garage entry).
- Cold start now shows an animated splash before Home.
- History header counts the selected filter range, not all-time.
